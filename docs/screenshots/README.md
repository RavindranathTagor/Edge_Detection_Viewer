# Screenshot Guide

## How to Capture Screenshots

### For Web Viewer
1. Open http://localhost:8080 in browser
2. Click "Load Sample Frame"
3. Take screenshot (Windows: Win + Shift + S)
4. Save as `web-viewer.png` in this directory

### For Android App
1. Build and install APK on device/emulator
2. Grant camera permission
3. Wait for edge detection to activate
4. Take screenshot:
   - Physical device: Power + Volume Down
   - Emulator: Camera icon in toolbar
5. Save as `android-app.png` in this directory

### Optional: Create GIF
Use tools like:
- **Windows**: ScreenToGif (free)
- **Cross-platform**: LICEcap, Peek
- **Online**: ezgif.com

Capture 5-10 seconds showing:
- Toggle between modes
- FPS counter updating
- Real-time processing

Save as `demo.gif` in this directory
